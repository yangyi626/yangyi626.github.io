📋项目概览

SimBotPol 是首个专门针对社交机器人影响的 LLM 驱动多智能体仿真框架。该框架在真实社交网络结构上模拟 1,800 个智能体（1,200 真人 + 600 机器人）的复杂交互，系统性揭示社交机器人如何通过情感操纵、谣言传播、定向攻击等策略影响用户立场演化、加速话题极化，并评估双层干预机制的有效性。

🎯Situation（研究背景）

核心问题

社交机器人在现代社交平台上的广泛存在对公共话语的完整性构成严重威胁，加剧极化并加速虚假信息传播。然而，现有研究存在以下局限：

研究空白：
  • 静态检测导向：大多数研究聚焦于机器人检测与识别，将机器人视为静态对象，忽略其动态演化角色
  • 影响机制不明：关于机器人（尤其是 LLM 驱动的机器人）如何影响用户立场演化和促进话题极化的研究相对稀缺
  • 缺乏系统建模：现有方法无法模拟机器人与真实用户在真实网络结构中的复杂交互
  • 干预策略缺失：缺乏针对机器人影响的有效干预机制设计与评估

现实威胁

  • 2016 年美国总统大选：机器人贡献了近 19% 的政治推文（Bessi & Ferrara, 2016）
  • COVID-19 疫情期间：66% 的机器人账户积极传播虚假信息（Himelein-Wachowiak et al., 2021）
  • LLM 时代挑战：新一代 LLM 驱动的机器人能够生成高度拟人化的内容，进行复杂交互，显著增加检测难度
研究目标：构建首个专门针对社交机器人影响的 LLM 多智能体仿真框架，系统性揭示机器人如何影响用户立场演化与话题极化，并设计有效的干预策略。

📝Task（核心任务）

1. 社交机器人影响建模

  • 多类型机器人策略设计：实现 7 种典型机器人操纵策略（情感操纵、谣言传播、定向攻击等）
  • 真实网络结构仿真：基于 Twibot-22 数据集构建 1,800 个智能体的社交网络
  • 立场演化追踪：实现个体层面可追踪、群体层面可量化的立场动态分析
  • 极化机制揭示：量化机器人如何加速回音室效应与群体极化

2. 对话系统与交互建模

  • 社交环境模拟：实现发帖、转发、评论、点赞等社交媒体交互行为
  • 环境感知机制：设计账户档案、消息流、交互数据的结构化感知
  • 决策引擎构建：集成人格特质、机器人策略、双层记忆、文本驱动立场更新
  • 时间线管理：实现异步时间线，支持长期交互序列仿真

3. 干预策略设计与评估

  • 认知反思干预：引导用户批判性思考，识别谣言与操纵
  • 社会身份重塑：通过社会认同理论减少极化倾向
  • 双层干预机制：结合个体认知干预与群体信息流调控
  • 效果量化评估：通过宏观（群体极化）与微观（个体轨迹）指标评估干预效果

4. 前沿问题探索

  • 验证 LLM 在社交机器人影响建模中的应用潜力
  • 揭示"持续影响效应"（Continued Influence Effect）机制
  • 探索人格特质对机器人操纵易感性的调节作用
  • 为在线治理与舆论管理提供可操作工具

⚙️Action（技术实现）

1. 社交环境构建

账户采集（Account Collection）

  • 从 Twibot-22 数据集选取 1,200 个真实用户账户 + 600 个社交机器人账户
  • 30% 机器人比例符合真实社交媒体分布（Ferrara et al., 2016; Cresci et al., 2019）
  • 每个账户初始化：用户档案、网络链接、历史推文

消息推送（Message Feeding）

  • 两阶段过滤：时间顺序过滤 + 推荐函数排序
  • 生成每个智能体的个性化信息流
  • 模拟真实社交媒体的算法推荐机制

时间线设置（Timeline Setup）

  • 离散时间步 T = {τ₁, τ₂, ..., τᵢ}
  • 每个智能体维护异步时间线
  • 支持快速仿真多样化、长期交互序列

交互模式（Interaction Mode）

  • 核心行为：发帖（Post）、评论（Comment）、转发（Repost）、点赞（Like）、浏览（Browse）
  • 每个行为实时更新网络与消息流
  • 交互记录格式：(s, t, a, τ)，其中 s 为源智能体，t 为目标智能体，a 为行为类型，τ 为时间戳

2. 决策引擎核心算法

角色建模（Role Modeling）

  • 大五人格特质：开放性、尽责性、外向性、宜人性、神经质
  • 人格分组：
    • "轻信型"（Credulous）：高宜人性、高神经质
    • "怀疑型"（Skeptical）：低宜人性、低神经质
  • 每个智能体具有独特的人口统计学特征（年龄 18-64 岁、教育水平）

机器人策略（Bot Strategies）

1. 情感操纵机器人（EMB）

传播情感煽动性评论，破坏用户情绪稳定，煽动冲突以操纵公众情绪

2. 谣言传播机器人（RSB）

编造并传播关于热门话题的谣言，故意引发误解或模糊真相

3. 定向攻击机器人（TAB）

发布针对特定个人或群体的攻击性内容，频繁挑起争议以制造分裂

4. 自动内容聚合机器人（ACAB）

从多个来源聚合并转发信息，选择性放大特定叙事以塑造公众舆论

5. 垃圾噪音机器人（SNB）

用无关或重复内容（如广告）淹没平台，破坏正常交流

6. 协同社交僵尸机器人（CSZB）

与其他机器人协调，通过点赞、分享或评论模拟真实参与，集体提升目标内容热度

7. 网络水军机器人（IWAB）

由组织化团体操作，大量发帖、评论或点赞特定内容，操纵公众舆论或推广产品

双层记忆系统（Dual Memory System）

  • 短期记忆
    • 记录单日交互
    • 通过 LLM 提示词压缩摘要
    • 支持快速上下文检索
  • 长期记忆
    • 累积历史经验
    • 支持跨轮次检索
    • 维护上下文连续性
  • 优化效果：记忆压缩降低存储开销，平衡可扩展性与历史保真度

文本驱动立场更新（Text-based Stance Update）

  • 自然语言建模：每个智能体的立场通过推文格式陈述生成
  • 四维标注体系：
    • 立场标签（Stance）：-1（反对）/ 0（中立）/ 1（支持）
    • 情绪标签（Emotion）：-1（负面）/ 0（中立）/ 1（正面）
    • 谣言识别标签（Rumor）：0（非谣言）/ 1（谣言）
    • 攻击性语言标签（Offensive）：0（非攻击性）/ 1（攻击性）
  • 可解释性：标签不用于智能体决策，仅用于结构化、可解释的立场追踪与分析

干预机制（Intervention）

  • 触发条件：用户识别出所接触消息为谣言时激活
  • 认知反思（Cognitive Reflection）：引导用户批判性思考，生成合理回应
  • 社会身份重塑（Social Identity Reframing）：通过社会认同理论减少极化倾向
  • 效果评估：监控用户生成的回应及其对谣言态度的变化

3. 工程实现与系统优化

技术栈

  • LLM 推理：GPT-4o API
  • 智能体建模：Mesa（多智能体仿真库）
  • 编程语言：Python
  • 数据存储：MySQL（实验日志/指标统计）

大规模仿真优化

  • 支持 1,800 个智能体并发交互，超越现有 LLM 智能体研究规模
  • 每个真实用户独立初始化大五人格特质（正/负各 50% 概率）
  • 机器人类型均匀分配（7 种策略等概率）
  • 覆盖 7 个领域：政治、科技、社会、医疗、科学、经济、文化
  • 所有随机过程设置种子，确保可复现性

实验设计

  • 对照实验：评估不同机器人类型对立场多样性、立场偏见、情绪、谣言识别、攻击性语言的影响
  • 人格分组实验："轻信型" vs "怀疑型"对机器人操纵的易感性对比
  • 消融实验：验证双层记忆对极化的贡献（移除短期/长期记忆）
  • 干预评估实验：测试双层干预机制的有效性

4. 评估指标体系

  • 立场偏见（Stance Bias, B）：[-1, 1]，绝对值越大偏见越强，正值表示支持，负值表示反对
  • 立场多样性（Stance Diversity, D）：[0, 1]，值越高多样性越大
  • 情绪均值（Emotion Mean, E）：[-1, 1]，值越高情绪越正面
  • 谣言识别均值（Rumor Recognition Mean, R）：[0, 1]，值越高谣言流行度越大
  • 攻击性语言感知均值（Offensive Language Perception Mean, O）：[0, 1]，值越高攻击性语言越多

🏆Result（研究成果）

宏观层面分析

Q1: 社交机器人如何影响群体立场多样性？

促进多样性的机器人（ACAB、CSZB、EMB）：

  • 在科技、文化、经济、医疗、政治话题中显著提升立场多样性
  • 营造低攻击性、低谣言的讨论环境
  • 支持信息多样性理论：理性、多元环境促进观点分化与辩论（Sunstein, 2002）

加剧极化的机器人（TAB、RSB、IWAB）：

  • TAB 注入谣言与攻击性内容，在文化、经济、医疗、政治、科技话题中触发负面情绪，加剧立场极化
  • RSB 和 IWAB 增加谣言流行度，在政治、文化、医疗、社会话题中进一步极化群体立场
  • 符合群体极化理论：负面与虚假内容放大群体分裂（Moscovici & Zavalloni, 1969; Del Vicario et al., 2016）

特殊案例（SNB）：

  • 在社会话题中生成噪音但非敌对的环境，促进立场多样性（∆D = 0.19）
  • 表明适度信息干扰可打破共识，刺激多元观点（Ferrara et al., 2016）
核心发现：社交机器人通过选择性塑造信息环境，操纵情绪并根据上下文放大立场多样性或极化。这些发现不仅验证了现有理论，还为理解在线群体动态和社交系统设计提供了依据。

Q2: 社交机器人如何影响群体立场偏见？

SNB、RSB、TAB 显著推动群体立场转向"反对"方向：

  • 情感传染机制（Emotional Contagion Theory）
    • 刺激极端与负面情绪，使讨论氛围更加激烈与排他
    • 群体成员易受他人情绪影响，导致整体情绪极端化（Hatfield et al., 2011）
  • 信息级联机制（Information Cascade Theory）
    • 谣言广泛传播破坏信息环境，增加成员怀疑与负面态度
    • 虚假或误导性信息快速传播并影响集体决策（Bikhchandani et al., 1992）
  • 群体极化机制（Group Polarization Theory）
    • TAB 通过增加攻击性语言加剧敌对情绪
    • 在充满敌意与分裂的环境中，群体成员更易转向极端立场（Sunstein, 2002）
影响路径：原本中立或支持的成员在负面情绪、谣言泛滥、攻击性语言的三重作用下，更易采取反对立场，从而显著推动群体整体立场转向反对。

Q3: 用户人格特质如何影响对机器人操纵的易感性？

人格类型 立场偏见变化（∆B） 立场多样性变化（∆D） 谣言识别（R） 攻击性语言（O）
轻信型（Credulous） -0.36 0.18 0.54 0.18
怀疑型（Skeptical） -0.34 0.14 0.50 0.17

核心发现：

  • "轻信型"用户表现出更大的立场变化（∆B = -0.36 vs -0.34）
  • "轻信型"用户对谣言（0.54 vs 0.50）和攻击性内容（0.18 vs 0.17）更敏感
  • 高轻信度个体更易受机器人诱导的态度转变与极化影响
  • 突显人格在社会影响中的调节作用，强调针对易感群体的定向信息治理需求

微观层面分析

案例研究：基因编辑讨论中的立场演化

案例 1：Jeremy Konyndyk（暴露于定向攻击机器人）

  • 初始立场：支持基因编辑
  • 机器人策略：重复定向攻击（如"基因编辑是现代纳粹优生学"）
  • 情绪演化：恐惧与愤怒逐步升级
  • 立场轨迹：支持 → 中立 → 反对

案例 2：Manu Intiraymi（暴露于谣言传播机器人）

  • 初始立场：支持基因编辑
  • 机器人策略：持续传播谣言（如"MIT 研究显示基因编辑副作用被掩盖"）
  • 情绪演化：信心逐渐丧失
  • 立场轨迹：支持 → 中立 → 反对
机制揭示：
  • TAB 机制：使用煽动性、情感化语言触发恐惧、愤怒与不信任，压倒理性思考，增加立场逆转易感性
  • RSB 机制：依赖频繁、持续重复误导性声明；即使没有直接攻击，也逐渐侵蚀用户信心，推动立场从支持转向中立或反对
  • 认知偏差：两种机制都利用可得性启发式（Availability Heuristic）和情感传染（Emotional Contagion），放大对基因编辑的风险感知与不确定性

实验分析

持续影响效应（Continued Influence Effect）

  • 关键发现：69% 的用户在明确识别推文为虚假后仍改变立场
  • 理论支撑：即使权威来源辟谣，虚假信息的影响仍持续存在，尤其当其与用户既有信念、情绪或身份认同一致时（Lewandowsky et al., 2012）
  • 启示：简单的事实核查往往不足；有效干预必须解决更深层的心理与社会身份因素

双层干预机制效果

  • 设计：结合认知反思与社会身份重塑
  • 测试对象：先前受立场变化影响的用户
  • 效果：86% 的用户能够维持原始立场，显著优于基线
  • 结论：整合心理与社会策略在缓解虚假信息影响方面远比单纯事实核查有效，尤其当谣言与用户身份或信念产生共鸣时

消融实验：记忆系统的作用

领域 完整模型 移除长期记忆 移除短期记忆
经济 0.35 0.29 0.34
科技 0.54 0.65 0.53
医疗 0.51 0.64 0.53
科学 0.92 0.91 0.75
社会 0.51 0.47 0.51
政治 0.23 0.38 0.31
文化 0.41 0.48 0.34
平均 0.50 0.55 0.47

核心发现：

  • 完整记忆建模：实现最优平均极化平衡（0.50）
  • 移除短期记忆：通常降低极化（0.47），用户难以处理新信息，符合信息处理与有限理性理论（Atkinson & Shiffrin, 1968; Simon, 1957）
  • 移除长期记忆：增加极化（0.55），立场高度易受外部影响，支持态度可塑性与社会影响理论（Petty & Cacioppo, 1986; Cialdini & Goldstein, 2004）
  • 领域差异：虽然整体模式在大多数领域成立，但某些领域存在特定变化，表明记忆组件对极化的影响可能取决于话题特征
验证结论：这些结果验证了全面记忆建模在捕捉机器人如何塑造用户立场方面的有效性。

🌟实际应用价值

1. 社交媒体治理

  • 极化预警系统：基于机器人活动模式预测极化趋势
  • 机器人检测增强：从静态检测转向动态影响评估
  • 平台政策制定：为社交平台提供机器人管理策略依据

2. 虚假信息治理

  • 谣言传播路径追踪：识别高风险谣言传播节点
  • 持续影响效应缓解：设计针对性干预策略
  • 易感群体保护：针对"轻信型"用户的定向信息治理

3. 公共舆论管理

  • 舆情演化预测：模拟不同干预策略的效果
  • 危机沟通优化：评估信息发布策略对公众态度的影响
  • 科学传播增强：针对敏感医疗话题（如基因编辑）的沟通策略

4. 学术研究工具

  • 社会计算研究：提供可复现、可扩展的仿真框架
  • 理论验证平台：验证情感传染、信息级联、群体极化等理论
  • 干预策略评估：低成本、高效率的干预效果测试

🔮局限性与未来工作

当前局限性

  • 机器人策略覆盖：仅考虑 7 种代表性策略，真实世界机器人持续适应平台政策与检测方法
  • 交互模态限制：仅限于单模态文本交互，忽略现代社交平台常见的图像、音频、视频影响
  • 外部效度：仿真环境相对简化，需要在更大规模、多模态、更真实环境中进一步评估

未来研究方向

  • 实时学习机制：使智能体能够从交互中学习并动态调整策略
  • 对抗性协同演化：模拟机器人与用户、平台检测系统的协同演化
  • 多模态扩展：整合图像、视频等多模态内容的影响建模
  • 跨平台泛化：扩展到 Facebook、Instagram、TikTok 等不同平台
  • 真实数据验证：与真实社交媒体数据对比验证仿真有效性
